##How To
###Build and Install
* `$ sudo aptitude install qt4-qmake libqt4-dev libqtwebkit-dev libqjson-dev libmysqlclient-dev`
* `$ cd ignsdk-debian`
* `$ qmake`
* `$ make`
* `$ sudo make install`
